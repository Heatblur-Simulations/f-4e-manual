# Contrôles

Le cockpit est entièrement interactif ; chaque interrupteur, bouton et molette peut être
manipulé.

De plus, MSFS permet de lier les commandes couramment utilisées directement aux
matériel, tel que des joysticks ou des manettes.

> 🚧 Exemple d'image de l'interface utilisateur des contrôles.

Le tableau suivant montre quelles liaisons communes sont connectées à quel contrôle dans
le poste de pilotage.

| Fonction | Liaison |
| ------------------------------------- | ----------------------------- |
| Coupure des gaz (gauche) | CONDITION LEVER 1 CUT OFF |
| Coupure des gaz (à droite) | CONDITION LEVER 2 CUT OFF |
| Papillon des gaz au ralenti (gauche) | CONDITION LEVER 1 LOW IDLE |
| Papillon des gaz au ralenti (droite) | CONDITION LEVER 2 LOW IDLE |
| Allumage du moteur (gauche) | SET TURBINE 1 IGNITION SWITCH |
| Allumage du moteur (droite) | SET TURBINE 2 IGNITION SWITCH |
| Extension des aérofreins | EXTEND SPOILERS |
| Rétraction des aérofreins | RETRACT SPOILERS |
| Speedbrakes Inc | INCREMENT SPOILERS |
| Speedbrakes Dec | DECREMENT SPOILERS |
| Bascule des aérofreins | TOGGLE SPOILERS |
| Déploiement du parachute de freinage | ARM AUTO SPOILERS |
| Largage du parachute de freinage | DISARM AUTO SPOILERS |
| Interrupteur principal du moteur (gauche) | SET ENGINE MASTER 1 |
| Interrupteur principal du moteur (droite) | SET ENGINE MASTER 2 |
| Interrupteur principal du moteur (gauche) – Bascule | TOGGLE ENGINE MASTER 1 |
| Interrupteur principal du moteur (droite) – Bascule | TOGGLE ENGINE MASTER 2 |
| Train rentré | GEAR UP |
| Train sorti | GEAR DOWN |
| Bascule du train | GEAR TOGGLE |
| Volets/Becs rentrés | FLAPS 1 |
| Volets/Becs sortis | FLAPS 2 |
| Volets/Becs sortis et abaissés | FLAPS 3 |
| Volets/Becs Inc | DECREASE FLAPS |
| Volets/Becs Dec | INCREASE FLAPS |
| Bouton NWS | SET STEERING |
| Levier de dégagement rapide d'urgence | SET AUTOPILOT DISENGAGE |
| Trim de tangage à cabrer | ELEVATOR TRIM UP |
| Trim de tangage à piquer | ELEVATOR TRIM DOWN |
| Trim de roulis à gauche | AILERON TRIM LEFT |
| Trim de roulis à droite | AILERON TRIM RIGHT |
| Trim de lacet à gauche | RUDDER TRIM LEFT |
| Trim de lacet à droite | RUDDER TRIM RIGHT |
| Chaleur Pitot | SET PITOT HEAT |
| Chaleur Pitot allumée | PITOT HEAT ON |
| Chaleur Pitot éteinte | PITOT HEAT OFF |
| Basculement de chaleur Pitot | TOGGLE PITOT HEAT |
| AFCS Allumer | AUTOPILOT ATTITUDE HOLD ON |
| AFCS Éteindre | AUTOPILOT ATTITUDE HOLD OFF |
| ALT Allumer | AUTOPILOT ALTITUDE HOLD ON |
| ALT Éteindre | AUTOPILOT ALTITUDE HOLD OFF |
| Pression altimétrique déc | INCREASE ALTIMETER |
| Altimètre Pression Inc | DECREASE ALTIMETER |
| Cap Bug Inc | INCREASE HEADING BUG |
| Cap Bug Dec | DECREASE HEADING BUG |
| Course Bug Inc | PFD COURSE INCREASE |
| Course Bug Dec | PFD COURSE DECREASE |
| Anti-givre moteur - Bascule | TOGGLE ANTI ICE |
| Antidérapant - Basculer | TOGGLE ANTI SKID BRAKES |
