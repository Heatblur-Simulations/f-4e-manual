# Définitions

Si un acronyme, tel que DSCG, n'est pas clair à un moment donné, recherchez-le dans la
liste exhaustive fournie au chapitre [Abréviations](../abbreviations.md).

La symbologie suivante est utilisée tout au long de ce manuel.

## Général

| Symbole | Signification | Descriptif |
| ------ | ------------------ | ------------------------------------------------------------ |
| 💡 | Remarque | Élément sur lequel une attention particulière est accordée |
| 🟡 | Attention | Doit être suivi pour éviter **les dommages à l'équipement** |
| 🔴 | Avertissement | Ignorer pourrait entraîner **des blessures corporelles ou la mort** |
| 🚧 | En construction | Une section est **en cours de réalisation** et sera encore améliorée |

## Listes de contrôle

| Symbole | Descriptif |
| -------- | ---------------------------------------------------------------------------------------- |
| **Gras** | L'étape doit être effectuée OU il s'agit d'une procédure en gras (les membres de l'équipage doivent être capables d'accomplir des procédures en gras sans référence à la liste de contrôle) |
| 🔧 | L'étape implique ou peut impliquer une interaction [crew chief](../crew_chief/overview.md) |
| ⚡ | L'étape **ne peut pas** être effectuée lorsque le **démarrage de la batterie** est effectué, une alimentation externe est nécessaire |
