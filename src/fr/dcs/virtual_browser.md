# Navigateur virtuel

Pour enrichir les études en vol de l'avion ou simplement pour surmonter certains
temps d'arrêt, un navigateur virtuel peut être ouvert (par défaut
<kbd>RCTRL</kbd>+<kbd>V</kbd>).

![Interface utilisateur du navigateur virtuel](../img/virtual_browser.jpg)

Le navigateur permet aux utilisateurs de

- regarder des vidéos tutorielles,
- lire la documentation fournie par des tiers,
- écouter une playlist musicale en vol,
- ou aussi jouer à des jeux par navigateur et bien plus encore.

L'interface utilisateur est intégrée au jeu et peut être redimensionnée et déplacée. Des boutons dans
le coin supérieur gauche permettent une navigation rapide.

> 💡 Pour autoriser la navigation libre, sélectionnez **Complet** pour le
> [Option spéciale](special_options.md#domain-access) _HB UI Domain Access_.

## Séance

Les données de session sont mémorisées et enregistrées localement dans un dossier comme

`C:\Users\John Doe\Saved Games\DCS_F4E\cache\`

La suppression de ce dossier réinitialisera toutes les préférences et paramètres du navigateur.

Si nécessaire, le niveau sonore du navigateur peut être contrôlé via le mixeur de volume
fourni par Windows via l'entrée **HeatblurUi.exe**.

![Contrôle du volume](../img/virtual_browser_volume_control.jpg)
