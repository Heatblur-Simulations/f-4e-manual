# Jester

Jester est entièrement ouvert au modding via une Lua-API. Toute caractéristique et fonctionnalité de
Jester
a été écrit en utilisant cette API.

![Jester](../../../img/jester_ai_code.jpg)

> 💡 Les concepteurs de missions peuvent appliquer un Jester non modifié.
> Voir chapitre [9.10. Éditeur de mission](../../mission_editor.md#allow-jester-modifications) pour plus de détails.

L’IA Jester se compose principalement de :

* logique - écrite avec [Lua-API](lua_api.md)
* interface utilisateur - sites Web intégrés pour la [Roue](wheel_ui.md) et les
  [Boîtes de dialogue](dialog_ui.md)
* sons - voir [Fichiers vocaux](voice_files.md)

> 💡 Faites-nous savoir si vous créez un mod Jester sympa qui, selon vous, devrait
> trouver sa place dans le module !
