# Manuel intégré

![Manuel intégré](../img/ingame_manual.jpg)

Ce manuel est intégré au jeu et peut être lu en vol.

Il offre également un moyen d'expliquer les commutateurs dans le cockpit directement en ouvrant le
manuel et en le faisant défiler jusqu'à l'endroit expliquant le commutateur correspondant.

## Contrôles

La liaison par défaut <kbd>M</kbd> peut être utilisée pour basculer vers le manuel. En le tenant
vers le bas en cliquant sur un interrupteur dans le cockpit, le manuel s'ouvrira automatiquement
et faites défiler jusqu'à la section expliquant ce commutateur.

De plus, les liaisons suivantes sont également disponibles :

- <kbd>RSHIFT</kbd> + <kbd>M</kbd> - ouvrir le manuel
- <kbd>RCTRL</kbd> + <kbd>M</kbd> - fermer le manuel
- <kbd>RALT</kbd> + <kbd>M</kbd> - basculer vers le manuel

> 💡 Afin de fermer le manuel, assurez-vous d'abord de lui retirer le focus clavier
> en cliquant n'importe où ailleurs dans le cockpit.
