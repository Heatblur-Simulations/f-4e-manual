## BIT 3 Poursuite angulaire

| Étape | Système | Actions |
|------|--------------------------------------------------------------------------------------------------------------|--------|
| 1. | Bouton de test | Set 3 |
|      | a. B-sweep et EL strobe à 0 ±0,2° (avant verrouillage).                                                         |        |
| 2. | Verrouillez la quatrième cible |        |
|      | a. Le point de visée tourne entre le cercle ASE et le cercle segmenté (ne dépassant pas plus de 1/4 de pouce). |        |
|      | b. Rmax et Rmin à 15 et 5 milles.                                                                          |        |
|      | c. Cercle ASE comme indiqué.                                                                                      |        |
|      | d. Le stroboscope EL descend de 30° (après verrouillage).                                                                   |        |
| 3. | Briser le verrouillage et verrouiller la 10ème cible |        |
|      | a. Le point de visée tourne à l’intérieur du cercle ASE.                                                                        |        |
| 4. | Maintenir le verrouillage pour BIT 4. |        |

| 2. | Verrouillez la troisième cible |                   |
|      | a. Lecture du taux de portée 0 ± 300 nœuds.                                 |                   |
|      | b. Visez le point à l’intérieur du cercle ASE.                                     |                   |
|      | c. Cercle ASE de 0,56 pouce de diamètre.                                    |                   |
| 3. | Briser le verrou |                   |
| 4. | Interrupteur à impulsion | SHORT |
| 5. | (P) Bouton du mode de visée | A/A |

|      | a. Le radar brise le verrouillage, la porte de portée balaie la cible suivante et se verrouille. |                   |
|      | b. Répétez cette étape pour toutes les cibles.                             |                   |
| 10. | Bouton air-air | PRESS |
|      | a. Lumière air-air | OFF |

## BIT 6 Affichage d'attaque et verrouillages AIM-7

| Étape | Système | Actions |
|------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------|
| 1. | Bouton de test | Set 6 |
|      | Le voyant T (piste) s'allume mais le voyant SKIN TRK reste éteint pendant ce test. Il s’agit d’une indication normale avec le DSCG.                                                                                                                                                                                              |                                   |
| 2. | Bouton d'alimentation du radar | TEST |
| 3. | Interrupteur d'alimentation des missiles | STBY |
| 4. | Stroboscope de plage de position | Au-delà de 15 milles.                  |
|      | a. Rmax à 15 ± 1,5 milles, Rmin à 5 ± 1,25 milles.                                                                                                                                                                                                                                                                       |                                   |
|      | Taux de portée 000 nœuds Indique les missiles/prises AIM -7E installés ou les stations vides. 900 nœuds indiquent que les missiles/bouchons AIM-7F sont installés. Après OFP P005, le numéro de version OFP sera affiché pendant les 10 premières secondes à la place de la lecture du taux de plage ; par exemple, la version OFP P005 sera d'environ 500 nœuds. |                                   |
| 5. | Déplacer le stroboscope vers le bas | (avec demi-action ou pleine action) |
|      | a. Visez le point stationnaire à l’intérieur de la boîte.                                                                                                                                                                                                                                                                                        |                                   |
|      | b. Voyant HOLD ALT allumé au-delà de 15 milles, éteint à 15 milles.                                                                                                                                                                                                                                                                   |                                   |
|      | c. IN RANGE et SHOOT s'allument lorsque le stroboscope de portée est compris entre Rmax et Rmin.                                                                                                                                                                                                                                              |                                   |
|      | d. Le cercle ASE reste de taille fixe.                                                                                                                                                                                                                                                                                     |                                   |
|      | e. La rupture X se produit, l'affichage du cercle ASE et du taux de portée disparaît, les voyants IN RANGE/SHOOT s'éteignent à 5 miles ou moins.                                                                                                                                                                                                            |                                   |
| 6. | Bouton d'alimentation du radar | STBY |
