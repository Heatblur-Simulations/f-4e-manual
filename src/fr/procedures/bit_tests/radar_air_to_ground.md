# Bits radar air-sol

## BIT 5

![manuel_radar_bit_5_air_to_ground](../../img/manual_radar_bit_air_to_ground_5.jpg)

### Pente de plage et zéro

| Étape | Système | Actions |
|------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------|
| 1. | (P) Bouton du mode de visée | A/G |
| 2. | Bouton d'alimentation du radar | TEST |
| 3. | Bouton de portée radar | AI 5 |
| 4. | Bouton de mode radar | A/G |
| 5. | Bouton de test | 5 |
|      | a. Moniteur (RDR MAG) 0,9 à 1,15 |        |
|      | b. Balayage B 5° ±3° à gauche du centre.                                                                                                                                                                                 |        |
|      | c. Symbole d'acquisition centré sur B-sweep (avant verrouillage).                                                                                                                                                        |        |
|      | d. La ligne d'horizon indique une inclinaison de 30° à gauche et une plongée de 30°.                                                                                                                                                             |        |
| 6. | Verrouiller la première cible |        |
|      | a. Le cercle ASE apparaît et le balayage B s'estompe (AGC). L'action AGC réduit l'intensité du balayage B et les cibles de test les plus faibles s'estompent. Cependant, il n’est pas anormal que les cibles de test les plus fortes soient visibles lors du balayage B. |        |
|      | b. (P) Barre de portée à 5h00 à ± 0,5° (6 000 pieds).                                                                                                                                                                 |        |
| 7. | Verrouiller la deuxième cible |        |
|      | a. (P) Barre de portée à 3h00 à ± 0,5° (12 000 pieds).                                                                                                                                                               |        |
| 8. | Verrouillez la troisième cible |        |
|      | a. (P) Barre de portée à 1h00 à ± 0,5° (18 000 pieds).                                                                                                                                                               |        |
