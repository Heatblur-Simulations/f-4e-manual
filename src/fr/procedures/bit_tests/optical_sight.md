# Test de visée optique

| Étape | Système | Actions |
|------|--------------------------------------------------------------------|---------------------|
| 1. | Pinky switch | RADAR |
| 2. | Levier de volet de visée | OPEN |
| 3. | Bouton du mode de visée | CAGE |
| 4. | Bouton de dépression du réticule | 35 MILS |
| 5. | Bouton d'intensité du réticule | AS REQUIRED |
| 6. | Commutateurs d'armement | OFF/SAFE |
|      | a. Feux verts/oranges de la station | OFF |
| 7. | Bouton de mode de livraison | DIRECT |
| 8. | Bouton de neutralisation de la sécurité de l'armement | PUSH IN |
| 9. | Bouton du mode de visée | CAGE à A/G à A/A |
|      | a. Pipper (à RBL) ne bouge pas de plus de ± 2 mils.                     |                     |
| 10. | Bouton de dépression du réticule | 10 MILS |
| 11. | Bouton du mode de visée | BIT 1 (figure 2-21) |
|      | a. Le réticule saute de 25 ± 4 mils à gauche.                                |                     |
|      | b. Les languettes du rouleau tournent à 90° dans le sens des aiguilles d'une montre.                             |                     |
|      | c. La barre de portée indique 4 000 pieds (3 heures à ± 15°.) |                     |
| 12. | Bouton du mode de visée | BIT 2 (figure 2-21) |
|      | a. Le réticule descend de 25 ± 4 mils.                    |                     |
|      | b. Les onglets de rouleau indiquent le vol en niveau.                           |                     |
|      | c. La barre de portée indique 6 700 pieds (12h30 à ±15°).      |                     |
| 13. | Bouton du mode de visée | A/A |
|      | a. Pipper revient à RBL.                                          |                     |
| 14. | (WSO) Bouton d'alimentation du radar | TEST |
| 15. | (WSO) Bouton de test | SET 5 |
| 16. | (WSO) Bouton de plage | AI 5 |
| 17. | (WSO) Bouton de mode radar | BST |
| 18. | (WSO) Verrouillage sur la première cible BIT |                     |
| 19. | Pinky switch | RADAR à HEAT |
|      | a. Pipper reste à RBL.                                          |                     |
| 20. | Pinky switch | GUNS |
|      | a. Lampe GUN tête haute | ON |
|      | b. La barre de portée indique 6 000 pieds (1 heure à ± 15°).          |                     |
|      | c. Le réticule s'abaisse lentement selon l'AOA correspondant : |                     |
|      | Unités AOA | (Mils) |
|      | 0                                                                  | (36)                |
|      | 5                                                                  | (31)                |
|      | 10                                                                 | (27)                |
|      | 15                                                                 | (22)                |
|      | 20                                                                 | (17)                |
|      | 25                                                                 | (13)                |
|      | 30                                                                 | (8)                 |
| 21. | Bouton cage | PRESS AND HOLD |
|      | a. Pipper se déplace jusqu'à une portée de cage de 1 000 pieds (environ 2 mils en dessous de RBL). |                     |
|      | b. La barre de portée reste à 6 000 pieds (1 heure à ± 15°).         |                     |
| 22. | (WSO) Briser le verrouillage radar |                     |
|      | a. Barre de portée | OFF |
|      | b. Pipper se déplace vers | RBL.                |
| 23. | Bouton cage | RELEASE |
|      | a. Pipper se déplace jusqu'à une portée de cage de 1 000 pieds (environ 2 mils en dessous de RBL). |                     |
| 24. | Bouton du mode de visée | STBY ou CAGE |
| 25. | Bouton de mode de livraison | OFF |
| 26. | Bouton de neutralisation de la sécurité de l'armement | RESET |
