# WRCS BIT

## WRCS BIT

| Étape | Système | Actions |
|------|-----------------------|--------------|
| 1. | INS | ALIGN ou NAV |
| 2. | Bouton cible alt/range | 170 |
| 3. | ARBCS minuteries de bombardement | SET |
|      | a. Pull-up | 5.0 |
|      | b. Libération | 10,0 |
| 4. | (P) Commutateur BRG/DIST | NAV COMP |
| 5. | (WSO) Commutateur de mode de navigation | NAV COMP |

### Mission de dépôt

| Étape | Système | Actions |
|------|------------------------------|---------------|
| 1. | Sélecteur WRCS BIT | LAYDOWN |
| 2. | Bouton BIT | PUSH and HOLD |
| 3. | Après 5 secondes, bouton FRZ. | PUSH ON |
|      | a. Indicateur de portée | ON |
|      | b. Après 15 secondes | GO/NO GO |

### Mission AGM-45

| Étape | Système | Actions |
|------|-----------------------------------------|-----------------|
| 1. | Sélecteur WRCS BIT | AGM-45 |
| 2. | Bouton BIT | PUSH and HOLD |
|      | a. BDHI et milles HSI 7,6 ± 1,0 NM |                 |
|      | b. Indicateur alternatif | ON |
| 3. | Après 5 secondes, bouton FRZ.            | PUSH ON |
|      | a. BDHI et miles HSI DIMINUTION |                 |
|      | b. Indexeur inférieur AOA | PULL UP COMMAND |
|      | c. Après 10 secondes, indexeur central AOA | LEVEL COMMAND |
|      | d. Après 5 secondes | GO/NO GO |
|      | e. Après 5 secondes, indexeur supérieur AOA | DIVE COMMAND |

### Mission de recherche de cible/compensation de bombe

| Étape | Système | Actions |
|------|----------------------------------------------|--------------------------------------------------|
| 1. | Sélecteur WRCS BIT | TGT FIND/OFFSET BOMB |
| 2. | Bouton BIT | PUSH and HOLD |
|      | a. Indicateur alternatif | ON |
| 3. | Après 5 secondes, contrôle du curseur le long de la piste | MOVE |
| 4. | Contrôle transversal | MOVE |
| 5. | Intensité du curseur | ADJUST |
| 6. | Bouton de réinitialisation | PUSH |
|      | a. Les curseurs reviennent à la plage zéro et à l'azimut. |                                                  |
| 7. | Bouton d'insertion de cible | PUSH ON |
|      | a. Curseur le long de la piste | 6,5 NM sur la ligne de quadrillage 00 (± 0,2000 pieds) |
|      | b. Curseur de piste croisée | 30° +1,5° DROITE |
|      | c. (P) Pointeur de roulement HSI | 23° ± 0,2,5° (à droite du poids) ; Lumière HSI TGT - ALLUMÉE |
|      | d. TGT INS lumière | ON |
|      | e. BDHI Aiguille n°1 | 23° ± 0,5° (à droite de l'index supérieur) |
|      | f. (P-WSO) HSI et BDHI milles | 4,8 ± 0,1,0 NM |
|      | g. (P) Onglets de viseur | ROTATED CLOCKWISE |
|      | h. (P) Pointeur vertical ADI | DEFLECTS FULL RIGHT |
| 8. | Après 5 secondes, bouton FRZ.                 | PUSH ON |
|      | a. (P-WSO) HSI et BDHI milles | DIMINUER JUSQU'À ZÉRO, PUIS AUGMENTER |
|      | b. (P) Onglets de viseur | TOURNE (lorsque les miles passent par zéro) |
|      | c. Après 15 secondes | GO/NO GO |

## Vérification de l'étalonnage et de l'intégration du champ de tir de la bombe

| Étape | Système | Actions |
|------|-------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------|
| 1. | Mode INS | NAV |
| 2. | Panneau de commande WRCS | SET |
|      | a. Distance cible N/S | N120 |
|      | b. Distance cible E/O | 000 |
|      | c. Altitude cible | SET PRÉSENT AVION HSL (29.92) |
|      | d. Libérer l'avance | 000 |
|      | e. Plage de sortie | 000 |
| 3. | Panneau de livraison d'armes | SET |
|      | a. Activer le commutateur | NORM |
|      | b. Commutateur de recherche de cible | HOLD |
|      | c. Commutateur de gamme | X100 |
| 4. | Commutateurs du cockpit avant | SET |
|      | a. Mode de livraison | T LAD |
|      | b. Sélection d'arme | BOMBS |
|      | c. Boutons de sélection de station | OFF |
|      | d. Annulation de l'armement | PUSH IN |
| 5. | Minuteries | SET (valeurs de livraison) |
|      | a. Minuterie de traction |                                                                                             |
|      | b. Minuterie de libération |                                                                                             |
| 6. | Bouton d'insertion de cible | PUSH ON |
|      | a. Curseur le long de la piste | Saute à la deuxième cible BIT |
| 7. | WRCS compteurs de distance N/S | Augmentez ou diminuez jusqu'à ce que les bords proches du curseur le long de la piste et la deuxième cible BIT coïncident. |
|      | a. Notez la différence entre le compteur et le N120.                                                                                        |                                                                                             |
|      | b. Pour une valeur de compteur plus grande, la correction est + ; pour une valeur de compteur plus petite, la correction est -.                                        |                                                                                             |
| 8. | Distance cible N/S | RESET |
|      | a. Réglez la plage de déclenchement du vol en amont ajustée pour l'erreur d'étalonnage à l'étape 7. |                                                                                             |
|      | b. Relâcher le commutateur de plage | NORM ou X100 (selon les besoins) |
|      | c. Activer le commutateur | ON |
| 9. | Compteur de plage de libération | AUGMENTER (lentement jusqu'à 1/3 de seconde de ton) |
|      | a. Vérifiez la précision de la minuterie.                                                                                                            |                                                                                             |
|      | b. (P) Vérifiez les indications LABS.                                                                                                      |                                                                                             |
|      | c. Notez la différence entre la lecture dérivée du compteur RR et la plage de libération/activation avant le vol et appliquez-la comme correction à toutes les plages. |                                                                                             |
|      | (1) Pour une valeur de compteur plus grande, la correction est + ; pour une valeur de compteur plus petite, la correction est de -.                                        |                                                                                             |
