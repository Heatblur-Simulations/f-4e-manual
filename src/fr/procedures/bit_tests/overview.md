# Procédures de test intégrées (BIT)

Ce chapitre contient toutes les procédures de test intégré (BIT) pour
le [système radar,](../../systems/radar/overview.md)
y compris [BIT air-sol](radar_air_to_ground.md)
et [BIT air-air](radar_air_to_air.md),
l'écran [DSCG](../../cockpit/wso/pedestal_group.md#digital-scan-converter-group-dscg) lui-même,
le [Pave Spike Pod de ciblage](../../systems/weapon_systems/pave_spike/overview.md), le [système
TACAN](../../systems/nav_com/tacan.md) et le [système VOR/ILS](../../systems/nav_com/vor_ils.md).
