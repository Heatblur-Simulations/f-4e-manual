# Essais de navigation

## Test TACAN

| Étape | Système | Actions |
|------|--------------------------------------------------------------------------------|-----------------------------------------------------------------|
| 1. | Bouton de sélection de fonctions | T/R |
|      | Prévoyez 90 secondes pour l'échauffement |                                                                 |
| 2. | Cours | RÉGLÉ À 180°.                                                    |
| 3. | Bouton test Tacan | PRESS AND RELEASE |
| 4. | Observez les points suivants : |                                                                 |
|      | a. Indicateur de test | Clignote momentanément.                                            |
|      | b. Drapeaux de parcours et de distance | En vue.                                                        |
|      | c. Pointeur de relèvement | Tourne à 270° pendant environ 7 secondes.                              |
|      | d. Drapeaux de parcours et de distance | Hors de vue.                                                    |
|      | e. Distance | Indique 0,0 +0,5 miles (-0,5 miles indiqué comme 399,5 miles). |
|      | f. Pointeur de relèvement | Pivote à 180° +3°.                                              |
|      | g. Indicateur d'écart de cap | À moins de 1/2 point du centre.                                     |
|      | h. Indicateur TO-FROM | Indique TO.                                                   |
|      | i. Après environ 15 secondes, les drapeaux de parcours et de distance sont visibles jusqu'à la fin du test. |                                                                 |

## Test VOR/ILS

| Étape | Système | Actions |
|------|------------------------------------------------------------------------------------------|---------------------------------------------------|
| 1. | ILS Sélecteur de fréquence | Sélectionnez la fréquence sur l'indicateur.                    |
| 2. | Appuyer sur le bouton-poussoir Test VOR/MKR | Les voyants de balise (marker beacon) s'allument.                  |
| 3. | VOR Fréquence sélectionnée | Doit être sélectionné avec un signal VOR valide présent. |
| 4. | Cours sélectionné | 315° sur le sélecteur de cap HSI.                  |
| 5. | Mode HSI | En VOR/ILS.                                       |
| 6. | Appuyez sur le bouton-poussoir de test | Lance le test.                               |
|      | a. L’indicateur d’écart de cap se déplace vers le centre (avec une erreur maximale autorisée de ±4°). |                                                   |
|      | b. L'indicateur « To-from » sur HSI indique TO.                                              |                                                   |
|      | c. Les voyants de balise (marker beacon) s'allument |                                                   |
|      | d. Les pointeurs de relèvement sur HSI et BDHI oscillent à 315° |                                                   |
