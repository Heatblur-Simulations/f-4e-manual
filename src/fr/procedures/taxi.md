## Avant le taxi

### Pilote

| Étape | Système | Actions |
| ------- | ------------------------------------- | --------------------------- |
| 1. | Équipement COMM & NAV | ON & CHECK                  |
| 2. | IFF | STANDBY                     |
| **3.** | **Altimètre radar** | **ON & CHECK**              |
| **4.** | **Altimètre et SPC** | **SET & CHECK**             |
| 5. | Aérofreins | CHECK🔧                    |
| 6. | Lattes et rabats | CHECK🔧                    |
| 7. | Commandes de vol | CHECK🔧                    |
| **8.** | **Lattes et rabats** | **NORM** |
| 9. | ARI Désengager | CHECK🔧                    |
| 10. | Commutateurs STAB AUG | ENGAGE AND CHECK🔧         |
| **11.** | **Sélection du système de référence** | **PRIM (INS in NAV)**       |
| 12. | Bouton de commande du mode boussole | SYNC                        |
| 13. | AFCS | CHECK (if required)         |
| 14. | Commutateurs STAB AUG | DISENGAGE                   |
| **15.** | **Garniture** | **CHECK🔧 & SET**          |
|         |                                       | **1-3 UNITS NOSE DOWN**     |
| 16. | Lattes & rabats | OUT & DOWN🔧               |
| **17.** | **Viseur optique** | **STBY/CAGE** |
| 18. | Pression pneumatique | CHECK                       |
| 19. | IFF | CHECK                       |
| 20. | Altimètre radar | CHECK & SET                 |
| **21.** | **Cales de roue** | **REMOVE**🔧               |

> 🟡 ATTENTION : Ne déplacez pas l'avion avec INS en mode ALIGN.

### WSO

| Étape | Système | Actions |
| ------ | --------------------- | --------------------- |

## Roulage

| Étape | Système | Actions |
|--------|------------------------|--------------------|
| 1. | Freins de roue | TEST               |
| **2.** | **Direction du train avant** | **ENGAGE & CHECK** |
| 3. | (P-WSO) Instruments de vol | CHECK          |
| 4. | (P-WSO) Dilueur d'oxygène | AS REQUIRED        |

> 🟡 ATTENTION : Roulez avec les verrières complètement ouvertes ou complètement fermées ; avec
> les verrières ouvertes, maintenir des vitesses de roulage inférieures à 60
> nœuds pour éviter d'endommager le mécanisme de manœuvre
> de la verrière.

> 🟡 ATTENTION : Une distance adéquate entre les avions doit être
> maintenu pendant le roulage en formation. Une verrière ouverte
> peut être endommagée par le souffle de l'échappement du moteur.

> 🟡 ATTENTION : Lors d'un roulage en cas de poids brut élevé
> conditions, le rayon de braquage doit être
> augmenté pour soulager les charges latérales excessives sur le
> les jambes de force, les roues et les pneus du train d'atterrissage principal.
