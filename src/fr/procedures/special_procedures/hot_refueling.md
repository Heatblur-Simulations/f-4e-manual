# Ravitaillement à chaud

Avant de faire le plein :

| Étape | Système | Action |
|------|------------------------------------------------------------------------------------------------------|------------|
| 1. | [Procédures de roulage sur monomoteur](single_engine_taxi.md) | COMPLETE |
| 2. | Internal wing transfer switch | NORMAL |
| 3. | External transfer switch | OFF |
| 4. | Air refuel switch | EXTEND |
| 4. | Après être entré dans la zone du hot pad, établissez un contact vocal (interphone avec le pilote et le superviseur de ravitaillement). |            |
| 5. | Refuel selection switch | AS DESIRED |

Pendant le ravitaillement :

| Étape | Système | Action |
|------|-----------------------------------------------------------------------------|--------|
| 1. | Restez attentif à tous les signaux visuels et vocaux du superviseur de ravitaillement. |        |
| 2. | Surveillez la fréquence des contrôles au sol.                                           |        |

Après le ravitaillement :

| Étape | Système | Action |
|------|-------------------------|----------|
| 1. | Air refuel switch | RETRACT |
| 2. | Refuel selection switch | INT ONLY |
