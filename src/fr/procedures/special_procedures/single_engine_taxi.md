# Taxi monomoteur

Si nécessaire, le moteur gauche peut être arrêté en premier.

| Étape | Système | Action |
|------|----------------------------|---------------|
| 1. | Right generator switch | OFF |
| 2. | BUS TIE OPEN light | OFF |
| 3. | Utility hydraulic pressure | WITHIN LIMITS |
| 4. | Right engine throttle | OFF |
