# Scramble

Les procédures de scramble suivantes supposent que
les actions suivantes ont été effectuées avant que l'avion ne
soit placé en état d'alerte (selon le type d'activités de scramble) :

* Inspection prévol complète incluant une inspection du cockpit sous
  tension, une vérification du fonctionnement du moteur et un contrôle
  opérationnel des aérofreins, des volets, des commandes de vol et du
  stab aug, conformément aux vérifications normales AVANT LE ROULAGE
  (COCKPIT AVANT).
* INS aligné et placé dans la mémoire de cap.
* L'avion est prêt pour le scramble conformément à la politique locale et
  aux instructions.

Si les actions ci-dessus ne sont pas terminées avant le scramble,
la procédure normale doit être utilisée.

## Avant le roulage (cockpit avant)

| Étape | Système | Action |
|------|----------------------------------------|------------------------|
| 1. | Communication and navigation equipment | ON AND CHECK |
| 2. | Emergency attitude indicator | SET |
| 3. | Altimeter & SPC | SET & CHECK |
| 4. | Takeoff trim | CHECK |
| 5. | Slats flaps | OUT & DOWN🔧 |
| 6. | Clearance to taxi from WSO |                        |
| 7. | Reference system selector | PRIM |
| 8. | Seat pins | CHECK REMOVED & STOWED |
| 9. | Personal equipment | CHECK |
| 10. | Wheel chocks | REMOVE🔧 |

## Avant le roulage (cockpit arrière)

| Étape | Descriptif | Action |
|------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------|
| 1. | Communication and navigation equipment | ON & CHECK |
| 2. | APX-80 mode switch | PASSIVE/OFF |
| 3. | Pressure altimeter | SET & CHECK |
| 4. | Radar power | STBY |
| 5. | Radar overtemp light | OFF, MONITOR |
| 6. | NAV Computer Mode | STBY |
| 7. | Heading memory alignment | COMPLETE |
|      | Si l'ensemble de navigation inertielle n'a pas été préalablement aligné pour la mémoire de cap, la référence d'attitude peut être obtenue en effectuant un alignement grossier (ALIGN - attendre 40 secondes - NAV). |                        |
| 8. | NAV Computer Mode | AS DESIRED |
| 9. | Seat pins | CHECK REMOVED & STOWED |
| 10. | Personal equipment | CHECK |
| 11. | Notify pilot | CLEAR TO TAXI |
