# Démarrage de la cartouche

Cette procédure est basée sur le fait que l'alimentation externe n'est pas
disponible. Pendant les démarrages à la cartouche de routine (autres que
l'alerte/exercice), il est conseillé d'utiliser l'alimentation électrique
externe, si disponible, car cela permettra l'utilisation du CNI, des circuits
d'avertissement de feu et de surchauffe, ainsi que de tous les instruments moteur
pendant le démarrage. Si une alimentation externe est utilisée, la commutation
normale du générateur doit être utilisée. Pour éviter d'éventuelles
irritation causée par la fumée/les gaz d'échappement de la cartouche, cela
il peut être conseillé de fermer les auvents et de sélectionner 100 % d'oxygène
pendant le cycle de démarrage.

> 🔴 AVERTISSEMENT : n'essayez jamais de charger ou de décharger une cartouche dans
> soit démarreur avec les coupe-circuits moteur allumés,
> interrupteur de ravitaillement au sol activé ou circuit électrique externe
> puissance appliquée. En cas de dysfonctionnement des circuits de sécurité
> se produit, des blessures graves pourraient en résulter.

| **1.** | **Engine master switches** | **ON** |
| **2.** | **Right generator switch** | **GEN ON** |
| **3.** | **Ignition button** | **PRESS AND HOLD** |
| **4.** | **Throttle** | **HALF THEN IDLE** |
| **5.** | **Engine start switch** | **R (right)** |
| **6.** | **At Lightoff (EGT INCR)** |                              |
|         | **Ignition Button** | **RELEASE** |
| 7. | Check Parameters |                              |
|         | a. Exhaust Gas Temperature | 220 - 420 °C |
|         | b. Idle RPM | 65 ± 1% |
|         | c. Right Boost Pump | 30 ± 5 PSI |
|         | d. Oil Pressure | 12 - 50 PSI |
|         | e. Fuel Flow Indicator | 800 - 1400 pph |
|         | f. Hydraulic Pressure | WITHIN LIMITS |
| 8. | Spoiler Actuator | CHECK (LEFT)🔧 |
| 9. | Air Refueling door | CHECK🔧 (if required) |
| **10.** | **Left Engine** | **START PER STEPS 1-7** |
| **11.** | **Left Generator** | **ON** |
| 12. | Right Generator | CYCLE OFF/ON |
| 13. | Bus Tie Open Light | OUT |
| 14. | Interior Check | COMPLETE |
