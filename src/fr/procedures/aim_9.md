# Missile AIM-9

Les listes de contrôle ci-dessous décrivent les procédures requises dans le poste de pilotage pour
l'emploi du missile AIM-9 Sidewinder. Afin d'avoir une meilleure compréhension des
méthodes d'emploi du missile, de la séquence de lancement et de ses variantes, veuillez vous référer à la
section [4.2.2 AIM-9 Sidewinder](../stores/air_to_air/aim_9.md).

## Lancement de l'AIM-9

### Pilote

| Étape | Système                                                               | Action              |
| ---- | -------------------------------------------------------------------- | ------------------- |
| 1.   | SIGHT MODE                                                           | A/A                 |
| 2.   | GUNS/MISSILE SWITCH                                                  | HEAT                |
|      | a. Head-up HEAT light                                                | ON                  |
| 3.   | MISSILE AURAL TONE                                                   | ADJUST              |
| 4.   | MASTER ARM                                                           | ARM                 |
|      | a. Head-up ARM light                                                 | ON                  |
| 5.   | UNCAGE SEEKER                                                        |                     |
|      | a. ARR BUTTON                                                        | PRESS AND HOLD      |
|      | b. REMARQUE : le chercheur n'est libéré de sa cage que lorsque le bouton ARR est enfoncé et maintenu | ON                  |
| 6.   | SHOOT LIGHTS                                                         | FLASHING            |
| 7.   | TRIGGER                                                              | SQUEEZE AND RELEASE |
