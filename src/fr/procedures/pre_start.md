# Préparation à l'inspection en vol et intérieure

> 💡 Si un démarrage sur batterie doit être effectué, les contrôles nécessitant une alimentation électrique devront
> être effectués après le démarrage des moteurs.

## WSO

### Avant l'alimentation électrique (WSO)

| Étape | Système                      | Action |
|------|------------------------------|--------|
| 1.   | Helmet                       | PUT ON |
| 2.   | AN/ALE-40 Chaff Dispenser    | OFF    |
| 3.   | Throttles                    | AFT    |
| 4.   | UHF radio                    | OFF    |
| 5.   | AVTR Switch                  | OFF    |
| 6.   | Radar power                  | OFF    |
| 7.   | DSCG                         | OFF    |
| 8.   | ECM equipment                | OFF    |
| 9.   | INS                          | OFF    |
| 10.  | Nuclear store consent switch | SAFE   |
| 11.  | Nav computer                 | OFF    |
| 12.  | Battery bypass switch        | OFF    |
| 13.  | Circuit breaker panels       | CHECK  |
| 14.  | Publications and flight data | CHECK  |

### Après l'alimentation électrique (WSO)

| Étape   | Système                             | Action         |
|--------|------------------------------------|----------------|
| **1.** | **Instrument ground power switch** | **ACTUATE**⚡   |
| 2.     | Navigation Computer                | SET            |
|        | **a. NAV Comp Mode**               | **STBY**       |
|        | b. Wind Counters                   | SET            |
|        | c. Variation Counter               | SET            |
|        | d. POS Update Switch               | NORM           |
|        | e. Present POS Counter             | SET            |
|        | f. Target Counters                 | SET FOR TGT 2  |
|        | g. NAV Comp Mode                   | RESET          |
|        | h. NAV Comp Mode                   | STBY           |
|        | i. Target Counters                 | SET FOR TGT 1  |
| **3.** | **INS Alignment**                  | **AS DESIRED** |

> 💡 Pour éviter les interruptions de courant électrique qui
> pourraient entraîner une indication INS NO-GO, assurez-vous que
> l'INS n'est pas en mode ALIGN lorsque les
> interrupteurs du générateur sont placés sur ON. Si une
> interruption de courant se produit, mettez le bouton de commande
> d'alimentation sur OFF. Lorsque le courant est rétabli, passez de
> OFF à ALIGN en marquant une pause momentanée sur STBY.

### INS Full Gyrocompass Alignment (WSO)

Le temps dépend de la température ambiante et de la précision de l'alignement BATH.

| Étape | Système         | Action   |
|------|----------------|----------|
| 1.   | NAV Comp Mode  | STBY     |
| 2.   | INS Power Knob | STBY     |
| 3.   | Gyro Heat Up   | WAIT     |
|      | a. HEAT Light  | OUT      |
| 4.   | INS Power Knob | ALIGN    |
| 5.   | Alignment      | WAIT     |
|      | a. ALIGN Light | FLASHING |
| 6.   | INS Power Knob | NAV      |
| 7.   | INS Light      | OUT      |

### INS Fast BATH Alignment (WSO)

Prend environ 2 minutes 15 secondes.

| Étape | Système         | Action |
|------|----------------|--------|
| 1.   | NAV Comp Mode  | STBY   |
| 3.   | INS Power Knob | STBY   |
| 4.   | INS Power Knob | ALIGN  |
| 5.   | Alignment      | WAIT   |
|      | a. HEAT Light  | IGNORE |
|      | b. ALIGN Light | STEADY |
| 6.   | INS Power Knob | NAV    |
| 7.   | INS Light      | OUT    |

### INS Stored Heading Alignment (WSO)
