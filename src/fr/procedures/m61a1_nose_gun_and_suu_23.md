# Canon nasal M61A1 et SUU-23

La liste de contrôle ci-dessous décrit les procédures requises dans le poste de pilotage pour
l'emploi du canon nasal M61A1 Vulcan et des pods de canon externes tels que le
SUU-16/A et le SUU-23/A. Afin de mieux comprendre leur
emploi, veuillez vous référer à la section [4.1 Armes à feu](../stores/guns.md).

## Pilote

| Étape | Système                    | Action            |
| ---- | ------------------------- | ----------------- |
| 1.   | SIGHT MODE                | A/G OR A/A AS REQ |
| 2.   | RETICLE DEPRESSION KNOB   | SET (A/G ONLY)    |
| 3.   | RATE SWITCH               | HIGH/LOW          |
| 4.   | DELIVERY MODE KNOB        | OFF               |
| 5.   | WEAPON SELECT KNOB        | NOT TV OR ARM     |
| 6.   | STATION SELECT BUTTONS    | GUN               |
|      | POUR LES PODS DE CANON :  |                   |
|      | a. STATION SELECT BUTTONS | SELECT STATIONS   |
| 7.   | POUR A/A :                 |                   |
|      | b. GUNS/MISSILE SWITCH    | GUN               |
| 8.   | TRIGGER                   | SQUEEZE           |
