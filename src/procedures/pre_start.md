# 飞行准备以及舱内检查

> 💡 如果使用电池起动，则必须在发动机起动后再进行需要电力的检查。

| 步骤 | 系统                 | 动作 |
| ---- | -------------------- | ---- |
| 1.   | 头盔                 | 戴上 |

| 3.   | 油门                 | 向后 |
| 4.   | UHF 电台             | OFF  |
| 5.   | AVTR 开关            | OFF  |
| 6.   | 雷达电源             | OFF  |
| 7.   | DSCG                 | OFF  |

### 通电后 (WSO)

| 步骤   | 系统                  | 动作         |
| ------ | --------------------- | ------------ |
| **1.** | **仪表地面电源开关**  | \*拨动\*\*⚡ |
| 2.     | 导航计算机            | 设置         |
|        | **a. 导航计算机模式** | **STBY**     |
|        | b. 风向风速计数器     | 设置         |
|        | c. 磁差计数器         | 设置         |
|        | d. 位置更新开关       | NORM         |
|        | e. 当前位置计数器     | 设置         |
|        | f. 目标计数器         | 设置到 TGT 2 |
|        | g. 导航计算机模式     | RESET        |
|        | h. 导航计算机模式     | STBY         |
|        | i. 目标计数器         | 设置到 TGT 1 |
| **3.** | **INS 对准**          | **按需**     |

> 💡 为避免断电可能导致出现 INS NO-GO 指示，确保 INS 在发电机开关拨至 ON 时未进入 ALIGN 模式。如果
> 出现电力中断，将电源控制旋钮转至 OFF 档位。当电力恢复后，重新从 OFF 转至 ALIGN 前中途请短暂停留在
> STBY 档位。

对准所需时间取决于环境温度和 BATH 对准精度。

| 步骤 | 系统             | 动作  |

|      | a. HEAT 灯       | 熄灭  |
| 4.   | INS 电源控制旋钮 | ALIGN |
| 5.   | 对准             | 等待  |
|      | a. ALIGN 灯      | 闪烁  |
| 6.   | INS 电源控制旋钮 | NAV   |
| 7.   | INS 灯           | 熄灭  |
